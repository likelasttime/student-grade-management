# 학생부 성적 프로그램
c++을 이용

### 주요기능
1. 학생 정보 입력
2. 학생 정보 출력
3. 학생 정보 수정
4. 학생 정보 삭제
5. 프로그램 종료


![프로그램 초기화면](https://user-images.githubusercontent.com/46569105/107872765-da154c00-6eef-11eb-960b-fd42a73db44c.PNG)

- 번호, 이름, 나이, 국어, 수학, 영어를 입력받음.
- 학생 정보 출력시 번호, 이름, 나이, 국어, 수학, 영어, 총점, 평균, 등수가 출력됨.
- 학생 정보를 수정 또는 삭제할 때 **번호**를 이용.
